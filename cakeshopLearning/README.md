## 下载后，需要添加数据库数据，并运行数据库（推荐使用elipse_javaee打开）
# 主要功能
* 用户登录注册
* 商品详情
* 购物车
* 订单查询
* 商品的增删查改
* 商品订单信息后台查看

# 使用的技术
* html
* css
* js
* jsp
* JDBC
* MySQL
