### vue-cms

这是一个根据某vue教学视频做的项目，实现有`新闻资讯`，`精品图片分享`，`商城功能`，看着视频比较火，就做了一下，由于视频是18年初左右的，现在webpack，Babel还有一些第三方组件都进行了更新，很多地方按照视频的做法会出现一系列问题，所以我将我的项目与解决方法上传上来供大家参考

### 使用

1. 在下载好的目录：

```shell
cnpm i
npm run dev
```

2. 或者直接以文件形式打开 ：双击运行下载好的dist/index.html文件，
